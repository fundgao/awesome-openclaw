**Sub2API：你需要知道的就这些**
GitHub 上有个项目叫 Sub2API，六千多 star，开源的。干的事儿说白了就是把你买的 Claude、OpenAI、Gemini 订阅，变成标准 API 接口，然后你想自己用也行，卖给别人也行。
多账号管理、Key 分发、按 Token 计费、自动调度——该有的都有。

搭起来不难：
- 一键装：bash curl -sSL https://raw.githubusercontent.com/Wei-Shaw/sub2api/main/deploy/install.sh | sudo bash
  装完浏览器打开 http://你的IP:8080，跟着向导走就行。
- 或者用 Docker：
  git clone https://github.com/Wei-Shaw/sub2api.git
  cd sub2api/deploy
  docker-compose up -d

五分钟能跑起来。
装好之后就四步：把你的订阅账号加进去，设好单价，生成 API Key，给客户用。完事。

**这行的坑，我一个不落地踩过**
封号
Anthropic 现在封杀越来越狠。请求体指纹、Machine ID 追踪、GA3 指纹检测全都上了。不是你会不会被封的问题，是早晚的问题。
