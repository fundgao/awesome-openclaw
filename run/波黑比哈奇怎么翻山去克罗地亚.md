# 波黑比哈奇怎么翻山去克罗地亚

15岁独自持中国护照游欧洲，从波黑到克罗地亚再到多国，如何应对突发状况还能顺利完成行程？

**靠主动沟通、借力当地善意和保持冷静，就能在意外中闯关成功，完成这段特别的旅行。**

我15岁那年独自闯欧洲，从波黑到克罗地亚再辗转多国，总结出我的独行应急攻略，现在分享给同样爱冒险的你。


方法1：遇边境荒路没住宿，找废弃建筑暂避，像我在克罗地亚就靠这招躲过雷雨；

方法2：主动和当地居民、牧师等交流，我在克罗地亚农户家获赠食物、德国教堂牧师帮我补水，都是靠主动开口；

方法3：遇到手续或行程问题，找当地官员求助，我在克罗地亚边境小镇就是靠官员帮忙才继续行程。
