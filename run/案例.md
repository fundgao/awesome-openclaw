# 案例1

跋山涉水两天三夜，冲出了比哈奇。

在上车点附近山里等到第二天凌晨上车，4:30分发车。

一直历经三次换乘，到达与斯洛文尼亚接壤边境。

穿过后坐大巴到达一个最近城市，走路到火车站坐上了到首都的列车。

到首都吃喝解决完继续出发到边境城市。

穿过后到达意大利，坐上了前往威尼斯的列车。

一路的压抑，辛苦，磕绊在此刻得到了释放。

路上穿过无数的荆棘丛，无数条小溪。横穿多条马路，有熊，有野猪，有鹿，夜里听到狼嚎觉都睡不着！

# 案例2

第一站，从北京飞到波黑，然后在卡塔尔的首都多哈转机去萨拉热窝。

到了萨拉热窝，应该是中午11点到12点左右，除了机场坐大巴到边境城市比哈奇。

到了比哈奇可以休整一晚上。

然后第二天再从比哈奇坐大巴车到边境小镇大克拉杜莎。

大克拉杜莎呢？前面有一条河，翻过这条河就到了克罗地亚境内的边境小镇卡尔洛维克。

到了卡尔洛维克以后，徒步去他的萨格勒布，也就是克罗地亚首都。

到了克罗地亚首都之后，选择乘火车前往克罗地亚，与斯洛文尼亚的边境城市萨默博尔。

到了萨摩博尔以后，直接就进入了斯洛文尼亚，因为都属于欧盟区边境，不设防，没有铁丝网，只有公路走路就过去了。

过去了以后找一个火车站坐到斯洛文尼亚的首都，卢卡雅尔纳。

从卢卡雅尔纳买火车票到意大利米兰。

再从意大利米兰坐大巴车到法国里昂。

到了法国里昂，可以玩一天到两天，然后乘车前往德国科隆。

德国科隆这个城市寻求庇护，因为难民集中营条件好。
